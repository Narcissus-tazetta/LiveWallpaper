# LiveWallpaper

[![macOS](https://img.shields.io/badge/macOS-12.0+-000000.svg?logo=apple&logoColor=white)](https://www.apple.com/macos/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

LiveWallpaperは、お気に入りの動画をMacのデスクトップ壁紙に設定できるアプリです。
他の作業をしている時のMacの負担（バッテリーやCPUの消費）をなるべく減らせるように、システムに配慮した設計にしています。


## 主な機能

* **動画を壁紙に**: 手元にある動画ファイル（mp4やmovなど）を壁紙として流せます。
* **マルチディスプレイ対応**: メイン画面だけ、もしくは接続しているすべての画面に表示するかを選べます。
* **プレイリスト**: 複数の動画を登録して、順番に再生することができます。
* **音量調整**: 動画の音量をアプリ内から調整可能です。
* **Macのための最適化設定**: 
  * 他のウィンドウが画面を大きく覆った時は、自動で壁紙の再生を一時停止します。
  * 「このアプリを使っている時だけは一時停止しない」というような除外設定も可能です。
  * フレームレート（30fps / 60fps / 制限なし）や、Macの負荷に合わせたデコードモードの設定ができます。
* **自動アップデート**: Sparkleを組み込んでいるので、アプリ内から簡単に最新版へアップデートできます。

## インストールと起動について

1. [Releases](../../releases) ページから、最新の `LiveWallpaper.app.zip` をダウンロードします。
2. ZIPファイルを解凍し、中に入っている `LiveWallpaper.app` を **「アプリケーション」フォルダ** に移動してください。

（※自動アップデート機能などを正常に動かすために、必ずアプリケーションフォルダに入れてから起動をお願いします。）

**【重要】初回起動時の注意点**
現在、Appleの公認署名を取得していないため、そのままダブルクリックで開こうとすると「悪質なソフトウェアかどうかをAppleでは確認できないため…」という警告が出て開けない場合があります。

その場合は、以下の手順で起動してください。
* 設定を開きます。
* プライバシーとセキュリティを開きます。
* セキュリティに移動して、「開く」を押します。
* 「このまま開く」をクリックしてパスワードを入力します。
* まだ開けていない場合、アプリケーションに行き、もう一度開きます。

## 使い方

アプリを起動すると、画面上のメニューバーにアイコンが追加されます。
そこから「設定」を開き、「壁紙」タブから好きな動画ファイルやプレイリストを追加してみてください。

## 動作環境

* macOS 12.0 (Monterey) 以降

## 開発について

このアプリは Swift と SwiftUI / AppKit を使って作られています。
ご自身でビルドする場合は、リポジトリをクローンしてXcodeで開き、依存関係（Sparkleなど）を解決してからビルドしてください。
